KETERANGAN

Tombol "Home" termasuk link relative apabila anda membuat suatu link pada page anda ke page lain pada komputer yang sama, tidak memerlukan menggunakan alamat internet yang lengkap. Jika kedua page pada direktori yang sama

Tombol "Chyr" termasuk link absolute yang melakukan link ke halaman web yang berada pada website lain (contoh nya link instagram, google, dll)

Tombol "kembali ke atas" termasuk Link pada halaman yang sama (link Anchor)
Jenis link ini digunakan ketika halaman web panjang, sehingga harus melakukan scroll berulang-ulang untuk membaca informasi dihalaman web tersebut. Untuk memudahkan navigasi dalam penelusuran halaman web antar bagian perlu dibuat link dan memberi nama sebagai alamat tujuan. Sehingga pada beberapa tempat di dalam dokumen akan ada bagian yang bernama, dan dibagian lainnya dapat diletakkan link untuk menuju bagian-bagian tersebut.
